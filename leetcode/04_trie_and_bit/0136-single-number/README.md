# 136. Single Number

## 我的思考
这题就是利用异或运算。

总结一下异或的关键性质：
1. 满足交换律和结合律  
2. 相同数字异或会抵消，`0` 不影响结果  

本质上，异或可以看作是**按位相加但丢失进位**：

- `1 ^ 1 = 0`（进位被丢弃）
- `1 ^ 0 = 1`（正常加法）
- `0 ^ 0 = 0`（正常加法）

因此数组中成对出现的数字会互相抵消，最终只剩下唯一的数字。

## 卡点
无

## 关键点
- 利用异或的交换律、结合律
- 相同元素异或为 0

## 复杂度
- 时间：O(n)
- 空间：O(1)
